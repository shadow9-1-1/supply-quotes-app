# Supply Quotes App V3 - Updated

## Run

```powershell
npm install
npm run dev
```

## Added

- PDF templates for AlexTrade, Gino Trade, Imdad, الوعد, and الحمد.
- Direct quote options for AlexTrade and Imdad, separate from their covered companies.
- PDF and Excel download buttons.
- Excel export with quote metadata, item table, unit price, totals, notes, and terms.
- Editable text lines above the table and separate editable terms below the table.
